# Tic Tac Toe – Java Swing

Petit projet Java permettant de jouer au Tic Tac Toe contre un ordinateur.  
L’utilisateur peut choisir la taille de la grille (3×3, 4×4, 5×5) et son pion (X ou O).  
L’application utilise **OpenJDK 21.0.2**.

---

## Exécution

### Depuis IntelliJ IDEA
1. Ouvrir le projet  
3. Lancer la classe **Main.java**

### Depuis un terminal
Depuis `src/main/java` :

javac Main.java ui/*.java logic/*.java
java Main
